Documentation

Welcome to the documentation for our latest template! Here you'll find all the information you need to get started with your new website. From installation to customization, we've got you covered. If you have any questions or issues along the way, don't hesitate to reach out to our support team.

Folder Structure

The template package you received contains 3 directories: demo, documentation, and template. Here's what each directory contains:


  • documentation/ - Start here to learn everything about the template and understand which file is responsible for what
  • demo/ - This is the demo page where you can experiment with different customizations like editing the title, description, colors, etc., and choose the design that best suits your needs
  • template/ - This is the main template which you'll use after finalizing your customizations. You'll need to apply the chosen configurations to this directory and then use it in your project

Customization Instructions

To customize the template, start by opening the "index.html" file located in the "demo" directory (demo/index.html) in your browser.


  • Select a color: In the top left corner, you'll find the "Select a Color" section, choose the color you'd like to use for your template
  • Set the Content: Below the color picker, you'll see the "Set the Content" section. Here, you can enable or disable the title and description.
  • Title: Click on the page title and enter the desired title for your coming soon template.
  • Description: Click on the description to write a brief explanation for your coming soon template.
  • Launch Date: This will be set in the main template folder.

Applying Customizations

It's now time to apply the customizations to the main template.

Open the index.html file located in the template directory (template/index.html) in your browser.

  • Apply Colors: Copy the HSL color values you selected from the demo page (without the percent symbol, just the numbers).
    Open the "style.css" file in the "template" folder (template/style.css) using a text editor.

    Replace the --h, --s, and --l color variables with the corresponding H-S-L colors you selected (e.g., --h: 264, --s: 100, --l: 61).

  • Apply Content: Open the "index.html" file located in the "template" folder (template/index.html) using a text editor.

    Title: At the bottom of the index.html file, you'll find a paragraph tag with an id of "title." Add your title content here or remove this paragraph if you don't need to display the title.

    Description: In a similar manner, you'll find a paragraph tag with an id of "description." Add your description here or remove this paragraph if you don't need to display the description.

  • Launch Date: Lastly, set the launch date in the "script.js" file located in the "template" folder (template/script.js).

    Open the file using a text editor and find the line under the comment "SET THE LAUNCH DATE HERE".

    Replace the Date('March 15, 2023 15:37:25') with the launch date of your coming soon website.

Publish Your Coming Soon Website

Once you have customized the template directory files, you are now ready to publish your website.

It is important to note that you only need to send the template/ directory with its files to your hosting provider. The demo/ and documentation/ folders were for the purpose of assisting you in customizing your website, and you don't need them for the deployment stage.
